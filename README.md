# 🚀 Trade Sphere – Modern Stock Trading & Portfolio Platform

**Trade Sphere** is a full-stack stock trading simulation platform inspired by modern trading apps.
It provides a seamless experience for tracking stocks, managing portfolios, and simulating trades with an elegant UI.

---

## ✨ Highlights

* 📊 Smart Dashboard with visual analytics
* 💼 Portfolio & Holdings management
* 🔄 Buy/Sell trade simulation system
* 📉 Positions tracking
* 🔐 User Authentication (Signup & Login)

---

## 🧩 Project Structure

```bash
Trade-Sphere/
│
├── backend/            # Node.js + Express server
│
├── dashboard/          # Trading dashboard (React)
│   ├── components/

│   └── charts & UI
│
├── frontend/           # Landing website (React + Vite)

│   │   └── support/
│
├── images/             # Screenshots & assets
```

## ⚙️ Installation

Install all dependencies:

```bash
npm install
```

---

## ▶️ Run the Project

### Backend

```bash
cd backend
npm start
```

### Frontend

```bash
cd frontend
npm run dev
```

### Dashboard

```bash
cd dashboard
npm run dev
```

---

## 📌 Usage

* Launch the app to open the **Landing Page**

* Navigate through different sections:

  * 🔑 Sign Up
  * ℹ️ About
  * 🛍 Products
  * 💰 Pricing
  * 🛠 Support

* Register a new account or log in with existing credentials

* Access the **Dashboard** to:

  * 📊 Track stock updates
  * 💼 Manage your portfolio
  * 🎮 Use the trade simulator

* 🔓 Log out anytime from the dashboard
